# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep the discussion focused on the work. Harassment
or exclusionary behavior isn't welcome here.

Report concerns to **mars@cruxia.ai**. Reports are handled confidentially.
